Linux命令行

初始shell

用户直接输入命令，Linux是理解不了的，Shell就会在中间充当翻译官的角色，把用户的命令翻译给Linux内核，然后Linux内核会执行命令。

用户就可以使用Shell，对Linux系统进行大部分的管理。

使用Shell分为两种方式：
1.交互式命令行，输入一行回车后执行一行的命令，适合量少的工作。
2.Shell脚本，输入多行命令，然后保存为文件，然后执行文件，适合批量工作或重复性的工作。

Bash Shell提示符

登录Linux系统后，默认进入交互式命令行模式，光标前面就会会看到提示符，提示符一般格式为:

用户名：当前登录的用户名
主机名：当前登录的计算机名称，修改主机名：

目录名：当前光标所在的目录，用户家目录表示成~
权限标识：超级管理员权限就表示成#，普通用户权限就表示成$

常见的命令

ls

用于显示当前文件下所含的内容（文件即子目录）

选项

选项的循序随便

  • -a:显示所有的隐藏文件
  • -l:显示文件的详细信息
  • -h:显示文件大小为可读性即将显示出来的文件大小以合适的单位显示出来
  • -F:显示文件类型，在文件名称后面加上文件类型标识符，例如可执行文档加*、目录加/、链接加@
  • -R:递归显示所有子目录下的文件，把目录下面的文件也列出来
  • -t:按照文件创建的时间排序
  • -r：将结果进行逆序排序

pwd

当前用户所在的目录

cd

用于当前切换工作目录
Linux里面的所有目录都是以/开头的

clear

清屏，快捷键Ctrl+L也可以

echo

用于字符串的输出

选项

  • -n:换行符，光标留在行尾
  • -e:转义字符，允许对下列出的加反斜线的转移字符进行解释
    – \a报警符
    – \b退格符
    – \f换页符
    – \n换行符
    – \r回车符
    – \t水平制表符
    – \v垂直制表符

系统命令

poweroff

关闭系统

reboot

重启系统

whoami

显示当前用户名

快捷键

  • Ctrl+C:终止当前正在执行的命令
  • Ctrl+D:退出当前用户，等价于exit
  • Ctrl+L:清屏
  • Ctrl+A:将光标移动到行首
  • Ctrl+E:将光标移动到行尾
  • Ctrl+K:删除从光标到行尾的所有字符
  • Ctrl+U:删除从行首到光标之间的所有字符
  • Ctrl+R:搜索历史命令，利用关键词

帮助命令

history

查看历史输入的命令

选项

  • -c:清除当前shell中所有历史命令
  • -r:从历史文件中读取命令
  • -w:将当前shell中的所有历史命令保存到历史文件中

使用！执行历史命令

  • !n:执行历史命令第n条
  • !string:从最近的命令中执行以string开头的命令
  • !-n:执行历史命令倒数第n条
  • ！！:执行上一条命令

help

查看命令的帮助信息

man

查看命令的帮助手册
-快捷键

按键 用途
空格键 向下翻一页
PaGe down 向下翻一页
PaGe up 向上翻一页
home 直接前往首页
end 直接前往尾页
/ 从上至下搜索某个关键词，如“/linux”
? 从下至上搜索某个关键词，如“?linux”
n 定位到下一个搜索到的关键词
N 定位到上一个搜索到的关键词
q 退出帮助文档

-手册的结构

结构名称 代表意义
NAME 命令的名称
SYNOPSIS 参数的大致使用方法
DESCRIPTION 介绍说明
EXAMPLES 演示（附带简单说明）
OVERVIEW 概述
DEFAULTS 默认的功能
OPTIONS 具体的可用选项（带介绍）
ENVIRONMENT 环境变量
FILES 用到的文件
SEE ALSO 相关的资料
HISTORY 维护历史与联系方式

alias

查看系统当前的别名
